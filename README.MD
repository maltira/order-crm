# OrderCRM

**Система управления заказами и клиентами**  
Лабораторная работа №2 по дисциплине «Системы управления базами данных»

---

## О проекте

Веб-приложение для управления клиентами, заказами и пользователями с ролевой моделью доступа.  
Реализовано в рамках учебной лабораторной работы с целью изучения взаимодействия backend-приложения с PostgreSQL.

### Основной стек технологий

- **Язык:** Go (Golang)
- **Фреймворк:** Gin
- **ORM:** GORM
- **База данных:** PostgreSQL
- **Авторизация:** JWT

---

## Возможности

### Роли и права доступа

| Роль        | Клиенты          | Заказы                    | Пользователи    |
|-------------|------------------|---------------------------|-----------------|
| **admin**   | Полный доступ    | Полный доступ             | Полный доступ   |
| **manager** | Полный доступ    | Создание, редактирование, оплата | Нет доступа  |
| **employee**| Только просмотр  | Создание заказов          | Нет доступа     |

### Функционал
- Авторизация / выход из системы
- Управление пользователями (только admin)
- Управление клиентами
- Создание и ведение заказов
- Добавление позиций в заказ
- Проведение платежей
- Изменение статусов заказов
- Разграничение доступа по ролям
